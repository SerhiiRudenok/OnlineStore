Як отримати з GitHubу те що хтось уже завантажив:

git pull origin main


Якщо в цьому завантаженні будуть нові моделі, то друга команда:

python manage.py migrate





----------------------------------------
Як завантажити свою інформацію на GitHub:
1. Додай зміни до індексу
2. Впиши коментар
3. Відправ на GitHub


git add .
git commit -m "Впиши свой комментарий "
git push
